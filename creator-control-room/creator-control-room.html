Creator Control Room

Analytical dashboard to see and keep track of live Twitch events and trends. Sign on creators using Twitch auth, and using an interactive dashboard, see events like new gifts, followers and subscribes in real time.

Tech Stack

TypeScript React Socket.io Docker S3 PostgreSQL
Live Demo

Features

Dashboard

The dashboard is a real time dashboard that shows the events happening on the stream such as new gifts, followers and subsribes.


This is also a interactive dashboard where users can filter by creator.


Also shows the stream in real time.




Easy Sign On

Users can sign on to the dashboard by using their Twitch credentials scanned through QR code.




How it works

Frontend:

Next.js (React/TypeScript) SPA/SSR hybrid, with a Socket.IO client for realtime updates. ￼

Backend (Cloud Run):

Node.js + Express for REST endpoints, plus a Socket.IO server that fans out events to tenant/creator “rooms” so each company only sees its own traffic. Cloud Run gives you managed HTTPS, autoscaling, and zero-infrastructure fuss in one containerized service. ￼

Realtime scale-out:

Socket.IO’s Redis adapter (e.g., Memorystore/Upstash) lets multiple Cloud Run instances broadcast consistently. ￼

Data:

PostgreSQL as the operational “hot” store (companies, creators, events) with Row-Level Security (RLS) enforcing tenant isolation; JSONB fields hold raw event payloads.￼

Assets/archives:

Object storage (e.g., brand logos) with pre-signed uploads; long-term event history can be archived cheaply for analytics. ￼
